0001-01-01 SW-FRAMEWORK-Keras

Concept

Keras is a high-level neural network API built on Theano or Tensorflow.
Compared with TensorFlow and pytorch and other packages, keras is more suitable for beginners who want to quickly experience machine learning

Because keras is a high-level neural network API, it is less suitable for some low-level changes, and it is more difficult to learn the principles behind neural networks in depth.
If you have the foundation of deep learning, or if you have more time, you can go to Tensorflow or pytorch for teaching

Using Keras as the interface of Tensorflow can be said to greatly reduce the entry threshold of neural networks
For beginners, it can greatly reduce the time to build a neural network, which is extremely convenient to use

Install
Install keras
sudo pip3 install keras
Install the model storage kit
sudo pip3 install h5py
=> So Keras' model file

Keras models

Keras has two types of models, sequential models and functional models. Functional models are more widely used.
=> Sequential model is a special case of functional model
Sequential model
=> Sequential model is a linear stack of multiple network layers, that is, "one road goes to black".
The model can be constructed by passing a layer list to the Sequential model
You can also add layers to the model one by one using the .add () method:
model = Sequential ()
model.add (Dense (32, input_shape = (784,)))
model.add (Activation ('relu'))
