0001-01-01 ALG-Makeup

Concept

主要写下后面涉及的美颜子项目及美颜的效果。

  • (1) 美白—肤色变白，饱和度，色调等
  • (2) 磨皮—美肤，皮肤光滑
  • (3) 滤镜—徕卡滤镜，ins滤镜
  • (4) 特效—抖动，灵魂出窍，马赛克特效
  • (5) 美形—瘦脸，大眼，瘦鼻，改眉形
  • (6) 美妆—美瞳，眼线，上妆
  • (7) 贴纸—脸萌
  • (8) 3d面具—套头，3d虚拟形象
  • (9) 人脸驱动—表情映射到另外一个人脸上
  • (10) 换脸—可以换喜欢的人脸
  • (11) 换发—发色可选择
  • (12) 人脸关键点识别—68，106，240人脸关键点级联识别，深度学习识别
    1-3涉及图像处理，
    4-9涉及图形学、渲染等，
    10-12涉及机器学习、深度学习。
    其中不是独立的，各个模块直接的知识相互交叉。
    => 所以和super resolution感覺不同
    => makeup (Perceptual Image Enhancement) 目標不是儘量還原原圖,而是做出不同的效果(fake),但不被發現
