0001-01-01 SW-FRAMEWORK-Android-AOSP

針對android 原本的AOSP的android code要怎麼download
https://source.android.com/setup/build/downloading

preparation

Installing Repo

Make sure you have a bin/ directory in your home directory and that it is included in your path:
mkdir ~/bin
PATH=~/bin:$PATH Download the Repo tool and ensure that it is executable:
curl https://storage.googleapis.com/git-repo-downloads/repo > ~/bin/repo
chmod a+x ~/bin/repo

Initializing a Repo client

After installing Repo, set up your client to access the Android source repository:會需要填寫好user資訊
mkdir Android-AOSP
cd Android-AOSP

git config –global user.name “evansin100”
git config –global user.email “evansin@gmail.com”

bring down the latest version of Repo with all its most recent bug fixes 把最新的code資訊抓下來
repo init -u https://android.googlesource.com/platform/manifest

可以指定要抓那一個版本Tag
To check out a branch other than “master”, specify it with -b.
For a list of branches, see Source Code Tags and Builds.
https://source.android.com/setup/start/build-numbers.html#source-code-tags-and-builds

例如這邊我是抓最新的 android 9.0 (就是P版) repo init -u https://android.googlesource.com/platform/manifest -b android-9.0.0_r30

Downloading the Android source tree

這樣就可以開始抓code了
pull down the Android source tree to your working directory from the repositories as specified in the default manifest
repo sync

build steps

Installing the JDK

sudo apt-get update
sudo apt-get install openjdk-8-jdk

build command

source build/envsetup.sh
Choose a target準備要build那個target
lunch aosp_arm64-eng
開始build
make -j4
